# Tron: Legacy Landing Page

An immersive **Tron: Legacy** inspired landing page focused on delivering a cinematic and interactive user experience.  
The project combines motion, sound, smooth navigation, and visual effects to recreate the digital universe of the film.

## Features

- **Interactive hero section**:
  - Dynamic grid background inspired by the Light Cycle sequence
  - Mouse movement controls horizontal motion, simulating the bike ride experience
- **Smooth and immersive navigation** powered by Lenis
- **Scanline screen effect**, simulating a retro computer / arcade display
- **Typed titles** using React Typed with a blinking cursor effect
- **Film information sections**:
  - About the movie
  - Soundtrack highlights
  - Main characters with short biographies
  - Photo gallery
  - Fun facts carousel
  - The legacy of *Tron: Legacy*
- **Music player**:
  - Plays *End of Line* by Daft Punk on user interaction
- **Navigation bar** with direct access to:
  - About
  - Gallery
  - Legacy

## Technologies Used
- TypeScript
- React
- Vite
- Tailwind CSS
- Three.js
- Lenis (smooth scrolling)
- AOS Animations
- React Typed
- Embla Carousel
- Vercel (deployment)
